# ComposeLab

A playground for learning Jetpack Compose and Material 3 on Android.

This is a personal learning project — experimenting with Compose fundamentals, state management, lists, and gestures by building a simple TODO app.

## What's in here

- Material 3 theming
- `LazyColumn` with `ViewModel` and `mutableStateListOf`
- Text input with `OutlinedTextField`
- Swipe-to-dismiss and swipe-to-mark-important gestures (WIP)

## Running

Open in Android Studio and run on a physical device or emulator (min SDK 26).
